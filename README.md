# Description
- The OpenWrt Firmwares for Amlogic S905x3
- Default IP: 192.168.0.254
- Default PW: password

## Install to EMMC
```
./install-to-emmc.sh
```
OR
```
cd /root && ./install-to-emmc.sh
```
